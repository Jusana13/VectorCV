# 📄 VectorCV — Editor Modular de Currículums A4

**VectorCV** es una aplicación web interactiva de alto rendimiento diseñada para la creación, personalización y previsualización de currículums en tiempo real. 

El proyecto destaca por su enfoque en la **modularidad extrema** y la **escalabilidad encapsulada**, permitiendo incorporar nuevas plantillas de CV y características de la interfaz de manera 100% declarativa, sin acoplamiento ni necesidad de modificar la lógica del motor principal.

## 🛠️ ¿Cómo funciona?

1. **Selección de Plantilla**: Elige una de las múltiples plantillas autocontenidas y declarativas de la colección.
2. **Edición Dinámica**: Introduce tus datos personales, profesionales y académicos en el panel izquierdo. Los cambios se sincronizan en tiempo real en la vista previa.
3. **Personalización**: Modifica colores, tipografías y la forma de la foto con controles dinámicos que se adaptan a las capacidades configuradas para cada plantilla.
4. **Exportación Vectorial**: Descarga tu currículum en formato PDF de alta calidad y texto seleccionable usando la función de impresión nativa del navegador.
